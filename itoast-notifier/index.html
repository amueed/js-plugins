Installation

Javascript

I have included 'itoast.min.js'. Just include this on your html page like this: 

<script type="text/javascript" src="js/itoast.min.js"></script>
jQuery or any other javascript library may be used with iToast, but is not required.

CSS

I have included 'itoast.css'. Just include this on your html page like this: 

<link type="text/css" rel="stylesheet" href="css/itoast.css"/>

Icons

I have included a libary called Font Awesome . It really is awesome! This is what will allow you to have 220 possible different icons on your notifications. If you aren't already using font awesome on your project then you can install it by simply including the two font-awesome css files like this: 

<link rel="stylesheet" type="text/css" href="css/font-awesome.css">
<!--[if IE 7]><link rel="stylesheet" href="css/font-awesome-ie7.css"><![endif]-->

Examples

Basic Usage

The most simple way to use iToast is to call the 'showNotification' function: 

iToast.showNotification('Notification','This is a simple notification');

Specifying Themes

There are also different theme styles that you can use:

  • 'default' - this is the default when you don't specify one
  • 'asphalt' - looks good on a light background
  • 'jetblack' - also looks good on a light background
  • 'frosted' - looks good on a dark background
  • 'square' - just for fun
  • 'mono' - just for fun
Here is how you can use them: 
iToast.showNotification('Notification','This is a simple notification',{ theme: 'asphalt' });

iToast.showNotification('Notification','This is a simple notification',{ theme: 'jetblack' });

Using Icons

Font Awesome will allow you to put one of 220 icons on your notificaton. All you have to do is include the font awesome class name, but without the "icon-" prefix. To see a list of all 220 icon class names, see the Font Awesome project home page . So this allows us to include an icon like this: 

iToast.showNotification('Notification','This is a simple notification',{icon: 'comment'});

You can also specify any color for your icon. Just specify a css color like this: 

iToast.showNotification('Notification','This is a simple notification',{icon: 'comment', iconColor: 'green'});
Or you can also use a hex value of course: 
iToast.showNotification('Notification','This is a simple notification',{icon: 'comment', iconColor: '#ffffff'});

Theming

You can easily create your own theme that can be accessed whenever you create a notification. Here is an example: 

/** this theme is called 'foobar' and can be specified
    when creating a notification because it has the class
    'theme-foobar'
**/
.itoast-main-container.theme-foobar .itoast-inner-container {
    background-color: #0262CC;
    opacity: .8;
    color: white;
    text-shadow: 0px 1px 2px rgba(256,256,256,1);
}
So to create a notification that uses your new theme, all you would have to do is include it as the theme property in your configuration like so: 
iToast.showToast('Message here',{theme: 'foobar'});

Notification Types

There are four different notification types that can be used as an alternative to the default options:

Success

Displays a green checkbox next to the message to give user feedback when they have successfully done something. Here's how you do it: 
iToast.showSuccess('Success','This is a simple message');

Warning

Displays a yellow warning sign next to the message to give user feedback when they need to be warned of something. Here's how you do it: 
iToast.showWarn('Warning','This is a simple message');

Error

Displays a red error sign next to the message to give user feedback when they need to be aware of an error that has occurred. Here's how you do it: 
iToast.showError('Error','This is a simple message');

Cool

Displays a blue thumbs up next to the message to give user feedback when they have done something cool. Here's how you do it: 
iToast.showCool('Cool','This is a simple message');

API

This is where iToast gets really fun. iToast is really flexible and allows you to do whatever you need.

showToast Function

iToast.showToast(message,options);
  • message:string - The message that will display in the body of the notification.
  • options:object - A configuration object containing each of the options for the notificaton.

Options Configuration Object

iToast.showToast(message,{
        title: undefined,
        icon: 'certificate',
        position: 'bottom right',
        pauseTime: 4000,
        theme: 'default',
        showCloseButton: true,
        pauseOnHover: true,
        returnAfterHoverTime: 1000,
        type: 'general',
        iconColor: 'none',
        onClick: function(e) {},
});
  • title:string - default:undefined - Specify an optional title heading that will show on the top of the notification in bold text.
  • icon:string - default:'certificate' - The icon that will appear next to the message. You can specify any icon from Font Awesome but without the "icon-" prefix.
  • position:string - default:'bottom right' - A string specifying where the notification will appear. possible options are:
    • 'top right'
    • 'bottom right'
    • 'top left'
    • 'bottom left'
  • pauseTime:int - default:4000 - The time in milliseconds to wait before the notification automatically goes away.
  • theme:string - default:'default' - A string specifying the theme to be used. This value is a hook into the css, so you can create your own theme. The possible options are:
    • 'default'
    • 'asphalt'
    • 'jetblack'
    • 'frosted'
    • 'square'
    • 'mono'
  • showCloseButton:boolean - default:true - Set to false if you don't want the close button to appear.
  • pauseOnHover:boolean - default:true - Allows the user to hover their mouse over the notification without it going away after the specified amount of time. Set to false if you don't want this behavior
  • returnAfterHoverTime:int - default:1000 - If 'pauseOnHover' is true, this is the time in milliseconds to wait before the notification goes away after the user has hovered over it and moved their mouse away.
  • type:string - default:'general' - The type of notification to be displayed. The icon will change color according to the type of notification you want. The possible options are:
    • 'general' - monochrome
    • 'success' - green
    • 'cool' - blue
    • 'error' - red
    • 'warn' - yellow
  • iconColor:string - default:none - The color of the icon. This will override the 'type' setting. If this is set to 'none' it will do nothing. Otherwise you can set it to any valid css color property. (ie: '#0000FF', 'rgba(20,20,30,.5)')
  • onClick:function - default:does nothing - This is a callback that will allow you to do anything when the user clicks on the notification. The event is passed in as the first parameter. Here's an example: 
    iToast.showToast('Click me!',{
  onClick: function(e) {
    alert('you clicked the notification!');
  }
});
